# LR6_tree
# АВЛ-дерево и красно-чёрное дерево на Python

## Описание проекта

В проекте реализованы две структуры данных:
- **АВЛ-дерево** — двоичное дерево поиска, в котором для каждого узла разность высот левого и правого поддеревьев не превышает 1.
- **Красно-чёрное дерево** — сбалансированное двоичное дерево поиска, в котором баланс поддерживается с помощью цветов узлов и правил перекрашивания.

Обе структуры поддерживают основные операции:
- создание узлов;
- вставку элементов;
- удаление элементов;
- вывод дерева в прямом порядке.

## Структура кода

### АВЛ-дерево

Используются два класса:
- `AVLNode` — хранит ключ, ссылки на левого и правого ребёнка и высоту;
- `AVLTree` — реализует операции дерева.

Основные методы `AVLTree`:
- `get_height(node)` — получение высоты узла;
- `get_balance(node)` — вычисление баланса;
- `right_rotate(y)` — правый поворот;
- `left_rotate(x)` — левый поворот;
- `insert(root, key)` — вставка элемента;
- `get_min(node)` — поиск минимального элемента;
- `delete(root, key)` — удаление элемента;
- `preorder(root)` — прямой обход дерева.

### Красно-чёрное дерево

Используются два класса:
- `RBNode` — хранит ключ, цвет, ссылки на левого и правого ребёнка и родителя;
- `RBTree` — реализует операции дерева.

Основные методы `RBTree`:
- `left_rotate(x)` — левый поворот;
- `right_rotate(y)` — правый поворот;
- `insert(key)` — вставка узла;
- `fix_insert(k)` — восстановление свойств после вставки;
- `minimum(node)` — поиск минимального элемента;
- `transplant(u, v)` — замена одного поддерева другим;
- `delete(key)` — удаление узла;
- `fix_delete(x)` — восстановление свойств после удаления;
- `preorder(node)` — обход дерева.
